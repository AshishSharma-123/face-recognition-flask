# Face Recognition Flask App

A simple **Face Recognition Web Application** built with **Flask and OpenCV**.  
The system allows users to create a dataset, train a face recognition model, and recognize faces using a webcam.

This project demonstrates how **computer vision** and **web applications** can be combined to build a simple face recognition system.

---

## 🚀 Features

- User Signup and Signin
- Dashboard UI
- Capture face images using webcam
- Automatically create dataset
- Train face recognition model
- Recognize faces in real-time
- Simple Flask web interface

---

## 🧠 Technologies Used

- Python
- Flask
- OpenCV
- NumPy
- Pillow
- HTML
- CSS
- JavaScript
